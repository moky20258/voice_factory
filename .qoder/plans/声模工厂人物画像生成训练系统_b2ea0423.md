# 声模工厂人物画像生成训练系统

## 功能概述

实现从人物画像描述到声模训练的完整自动化流程：
1. 解析人物画像描述（预设或自定义）
2. 根据画像特征配置 TTS 参数
3. 使用 Fish Speech TTS 生成训练音频
4. 执行完整训练流程（DSP增强→质量筛选→RVC训练）
5. 自动添加到声模数据库和人物画像系统

## 实现步骤

### Step 1: 创建人物画像到 TTS 参数的映射系统

**文件**: `voice_factory/portrait_to_tts_config.py`

**功能**:
- 建立人物画像特征与 TTS 参数的映射关系
- 支持预设画像（20个已有画像）
- 支持自定义画像描述解析

**核心逻辑**:

```python
# 人物画像特征映射表
PORTRAIT_TTS_MAPPING = {
    "中年男中音": {
        "temperature": 0.75,  # 较低温度，声音稳定
        "top_p": 0.85,
        "repetition_penalty": 1.15,
        "seed_range": (1000, 3000),  # 特定种子范围产生男声
        "reference_voice": "male_reference.wav",  # 参考音频（如果有）
    },
    "青年女高音": {
        "temperature": 0.80,  # 较高温度，声音活泼
        "top_p": 0.88,
        "repetition_penalty": 1.12,
        "seed_range": (7000, 9000),  # 特定种子范围产生女声
        "reference_voice": "female_reference.wav",
    },
    # ... 其他画像映射
}
```

**关键实现点**:
- Fish Speech TTS 的 `seed` 参数会影响声音特征
- `temperature` 影响声音的稳定性（低=稳定，高=变化）
- `top_p` 影响声音的表现力
- 需要通过实验确定不同参数组合产生的声音特征

### Step 2: 创建文本生成器

**文件**: `voice_factory/portrait_text_generator.py`

**功能**:
- 根据人物画像生成适合的训练文本
- 不同画像使用不同风格和内容的文本

**示例**:
```python

"作为一名经验丰富的专业人士，我认为这个方案非常可行。"
```

**实现**:
- 为每种画像类型准备 50-100 句专属文本
- 文本风格匹配画像特征（年龄、性别、音色）
- 保存到 `texts/portrait_{画像类型}.txt`

### Step 3: 创建声音生成器

**文件**: `voice_factory/portrait_voice_generator.py`

**功能**:
- 接收人物画像描述
- 查询 TTS 参数配置
- 调用 Fish Speech API 生成音频
- 生成指定数量的音频文件（默认 30 句）

**核心流程**:
```python
def generate_voice_from_portrait(
    portrait: str,           # 人物画像描述
    output_dir: str,         # 输出目录
    num_sentences: int = 30  # 生成句子数
):
    # 1. 查询 TTS 参数配置
    tts_config = get_tts_config_for_portrait(portrait)
    
    # 2. 加载对应文本
    texts = load_texts_for_portrait(portrait, num_sentences)
    
    # 3. 调用 TTS API 生成音频
    for i, text in enumerate(texts):
        audio_data = call_tts_api(text, tts_config)
        save_audio(audio_data, f"{output_dir}/{i:04d}.wav")
    
    # 4. 生成 metadata
    save_metadata(portrait, tts_config)
```

**关键点**:
- 复用 `generate.py` 中的 API 调用逻辑
- 支持断点续传
- 支持错误重试
- 输出到 `outputs/portrait_{画像类型}/` 目录

### Step 4: 创建完整流程集成脚本

**文件**: `voice_factory/portrait_to_voice_model.py`

**功能**: 整合所有步骤，实现端到端自动化

**完整流程**:
```
输入人物画像描述
    ↓
[1] 解析画像，生成 TTS 配置
    ↓
[2] 生成训练文本
    ↓
[3] 调用 TTS 生成音频 (30句)
    ↓
[4] DSP 增强处理 (复用 dsp_enhancer.py)
    ↓
[5] 质量筛选 (复用 quality_filter.py)
    ↓
[6] RVC 训练 (复用 rvc_full_auto_train.py)
    ↓
[7] 添加人物画像到数据库
    ↓
[8] 更新声模匹配系统
    ↓
输出完整声模
```

**核心代码结构**:
```python
def portrait_to_voice_model(
    portrait: str,                    # 人物画像
    speaker_id: Optional[int] = None, # 指定speaker编号
    epochs: int = 50                  # 训练轮数
):
    # 1. 确定 speaker ID
    if speaker_id is None:
        speaker_id = get_next_speaker_id()
    speaker_name = f"speaker_{speaker_id:04d}"
    
    # 2. 生成声音
    print(f"[1/6] 生成 {portrait} 的声音...")
    output_dir = f"outputs/{speaker_name}"
    generate_voice_from_portrait(portrait, output_dir)
    
    # 3. DSP 增强
    print(f"[2/6] DSP 增强处理...")
    run_dsp_enhancement(output_dir)
    
    # 4. 质量筛选
    print(f"[3/6] 质量筛选...")
    filtered_dir = run_quality_filter(output_dir)
    
    # 5. RVC 训练
    print(f"[4/6] RVC 训练...")
    rvc_train_dir = prepare_rvc_training_data(filtered_dir, speaker_name)
    train_rvc_model(rvc_train_dir, speaker_name, epochs)
    
    # 6. 添加人物画像
    print(f"[5/6] 添加人物画像...")
    add_portrait_to_database(speaker_name, portrait)
    
    # 7. 更新匹配系统
    print(f"[6/6] 更新匹配系统...")
    update_matching_system(speaker_name)
    
    print(f"✅ 完成！声模 {speaker_name} 已训练完毕")
```

## 技术要点

### 1. TTS 参数与声音特征的关系

需要通过实验确定：
- 不同 `seed` 范围产生的声音特征（男/女，高/低音）
- `temperature` 对声音稳定性的影响
- `top_p` 对声音表现力的影响
- `repetition_penalty` 对声音自然度的影响

**建议实验方案**:
```python
# 测试不同 seed 范围
for seed_range in [(1000, 3000), (3000, 5000), (5000, 7000), (7000, 9000)]:
    generate_test_audio(seed_range)
    analyze_voice_features()
    # 记录结果，建立映射表
```

### 2. 参考音频的使用

Fish Speech TTS 支持参考音频（reference audio）：
- 如果有目标音色的参考音频，可以直接使用
- 参考音频会影响生成声音的音色特征
- 可以为每种画像类型准备参考音频

**实现**:
```python
# 使用参考音频
payload = {
    "text": text,
    "reference_audio": "references/male_middle.wav",
    "reference_text": "这是一段参考文本",
    "temperature": 0.75,
    # ...
}
```

### 3. 文本风格匹配

不同画像需要不同风格的文本：
- 年龄：用词习惯、语速暗示
- 性别：语气词、表达方式
- 音色：情感色彩、表达风格

**文本模板示例**:
```python
TEXT_TEMPLATES = {
    "中年男中音": [
        "作为一名{职业}，我认为{观点}。",
        "根据我多年的经验，{建议}。",
    ],
    "青年女高音": [
        "哇，{事物}看起来好{形容词}呀！",
        "今天真是{形容词}的一天呢~",
    ],
}
```

## 注意事项

1. **TTS 质量**: Fish Speech TTS 生成的音频质量影响最终声模效果
2. **参数调优**: 需要通过实验确定最佳的 TTS 参数映射
3. **训练时间**: 完整流程约需 10-30 分钟（取决于 GPU）
4. **存储空间**: 每个声模约占用 1-2 GB 空间
5. **API 依赖**: 需要 Fish Speech API 服务运行

## 后续优化

1. **参数自动优化**: 基于生成声音的特征自动调整 TTS 参数
2. **质量评估**: 自动评估生成声音的质量，不合格则重新生成
3. **参考音频库**: 为每种画像准备高质量参考音频
4. **批量优化**: 支持批量生成多个声模
5. **Web 界面**: 提供图形化操作界面
