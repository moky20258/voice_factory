# 🎨 人物画像声模生成系统使用指南

## 📋 功能概述

本系统实现了从人物画像描述到完整声模训练的端到端自动化流程：

1. **解析人物画像** - 支持预设画像和自定义描述
2. **生成 TTS 参数** - 根据画像特征自动配置
3. **生成训练音频** - 使用 Fish Speech TTS 生成
4. **完整训练流程** - DSP增强→质量筛选→RVC训练
5. **自动添加数据库** - 纳入声模匹配系统

## 🚀 快速开始

### 方式一：批处理文件（推荐）


### 3. TTS 参数说明

系统根据人物画像自动配置以下参数：

| 参数 | 说明 | 影响 |
|------|------|------|
| temperature | 采样温度 (0.73-0.82) | 低=稳定，高=变化 |
| top_p | 核采样参数 (0.83-0.89) | 影响表现力 |
| repetition_penalty | 重复惩罚 (1.11-1.16) | 影响自然度 |
| seed | 随机种子 | 决定声音特征 |

### 4. 种子范围映射

| 画像类型 | 种子范围 | 声音特征 |
|---------|---------|---------|
| 中年男低音 | 500-1500 | 深沉、磁性 |
| 中年男中音 | 1000-2500 | 温暖、稳重 |
| 青年男高音 | 2500-4000 | 明亮、活力 |
| 中年女低音 | 4000-5500 | 温婉、成熟 |
| 青年女中音 | 5000-7000 | 温柔、知性 |
| 青年女高音 | 7000-9000 | 清亮、活泼 |

## 💡 使用技巧

### 1. 提高生成质量

- **增加句子数**: `--sentences 50` (推荐 30-50 句)
- **调整训练轮数**: `--epochs 60-80` (根据音频质量)
- **使用 DSP 增强**: 集成 dsp_enhancer.py
- **启用质量筛选**: 集成 quality_filter.py

### 2. 批量生成

创建批处理脚本：


## 🔍 故障排除

### 问题1: API 连接失败

**错误**: `❌ 无法连接到 Fish Speech API`

**解决**:
```bash
# 检查 API 服务是否运行
python tools/api_server.py

# 测试 API
curl http://127.0.0.1:8080/
```

### 问题2: 生成音频质量差

**可能原因**:
- TTS 参数不合适
- 文本风格不匹配
- 种子选择不佳

**解决**:
- 尝试不同的种子范围
- 调整 temperature 和 top_p
- 使用更匹配的文本

### 问题3: 训练效果不好

**建议**:
- 增加训练音频数量（50+ 句）
- 增加训练轮数（60-100 epochs）
- 使用 DSP 增强处理
- 启用质量筛选
