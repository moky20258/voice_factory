## 项目概述

**声模工厂**是一个完整的自动化声音人格生成系统，能够：

✅ 批量生成基础声音（Fish Speech）  
✅ 智能质量筛选（Whisper ASR）  
✅ DSP 增强扩库（10x 扩库能力）  
✅ 声纹数据库建立（FAISS）  
✅ 自动推荐匹配（Resemblyzer）  
✅ RVC 声模训练（输出 .pth + .index）

### 项目结构

```
W:/
├── fish-speech-1.5.1/          # Fish Speech TTS
│   └── voice_factory/          # 声模工厂
│       ├── texts/              # 训练文本
│       ├── outputs/            # 原始音频
│       ├── filtered/           # 筛选后音频
│       ├── enhanced/           # DSP 增强音频
│       ├── embeddings/         # 声纹向量
│       ├── voice_database/     # FAISS 数据库
│       ├── generate.py         # 批量生成器
│       ├── quality_filter.py   # 质量筛选器
│       ├── dsp_enhancer.py     # DSP 增强器
│       ├── complete_pipeline.py # 完整流水线
│       └── recommend_voice.py  # 推荐系统
│
└── rvc/                        # RVC WebUI
    ├── infer-web.py            # RVC 启动脚本
    └── logs/                   # 训练数据
```

#### 5.5 训练参数建议

| 参数 | 推荐值 | 说明 |

| Batch Size | 8-16 | 根据显存调整 |
| Sample Rate | 40k | 与预训练模型匹配 |
| Save Frequency | 10 | 每 10 epoch 保存一次 |

## 常见问题

### Q1: Python 版本问题

**错误**：`ModuleNotFoundError: No module named 'pyrootutils'`

**解决**：使用 Python 3.12，不要使用 3.14
```bash
C:\Users\moqi\AppData\Local\Programs\Python\Python312\python.exe ...
```

### Q2: FFmpeg 未找到

**错误**：`❌ 未找到 FFmpeg`

**解决**：
```bash
# 安装 FFmpeg
winget install ffmpeg

# 刷新环境变量（重新打开终端）
```

### Q3: RVC 训练很慢

**原因**：使用 CPU 训练

**解决**：
- 使用 GPU 训练（NVIDIA 显卡）
- 减少 Epoch 数量（100-150）
- 减小 Batch Size（4-8）

### Q4: 推荐系统不准确

**原因**：查询音频太短或质量差

**解决**：
- 使用 5-10 秒清晰录音
- 避免背景噪音
- 使用正常说话音量

### Q5: 内存不足

**解决**：
- 减少同时处理的音频数量
- 使用 `--batch` 分批处理
- 关闭其他程序

## 高级用法

### 自定义生成参数

编辑 `generate.py`：

```python
# 稳定真人感
TEMP_RANGE = (0.6, 0.7)
TOP_P_RANGE = (0.6, 0.7)
REPEAT_PENALTY_RANGE = (1.2, 1.3)

# 批量生成新音色
TEMP_RANGE = (0.75, 0.85)
TOP_P_RANGE = (0.80, 0.90)
REPEAT_PENALTY_RANGE = (1.10, 1.20)

# 情绪化生成
TEMP_RANGE = (0.85, 0.95)
TOP_P_RANGE = (0.85, 0.95)
REPEAT_PENALTY_RANGE = (1.05, 1.15)
```

### 自定义 DSP 增强

编辑 `dsp_enhancer.py`：

```python
DSP_CONFIGS = {
    "pitch": [-3, -2, 2, 3],      # 音高调整（半音）
    "formant": [0.85, 0.9, 1.1, 1.15],  # 共振峰
    "rate": [0.9, 1.1],           # 语速
    "eq": {
        "studio": "equal=f=1000:t=h:w=2:g=-3",
        "warm": "equal=f=200:t=h:w=1:g=2,equal=f=5000:t=h:w=2:g=-2",
        "bright": "equal=f=3000:t=h:w=2:g=3,equal=f=200:t=h:w=1:g=-2",
    }
}
```

### 集成到 Web 应用

创建 Flask API：

```python
from flask import Flask, request, jsonify
import recommend_voice as rv

app = Flask(__name__)

@app.route('/api/recommend', methods=['POST'])
def recommend():
    audio_file = request.files['audio']
    audio_file.save('temp.wav')
    
    index, speaker_list, encoder = rv.load_recommender()
    results = rv.query_voice(index, speaker_list, encoder, 'temp.wav', top_k=5)
    
    return jsonify(results)

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000)
```
