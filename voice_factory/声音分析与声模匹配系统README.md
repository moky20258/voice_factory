# 🎤 声音分析与声模匹配建议系统

## ✨ 系统简介

这是一个智能化的声音分析和声模匹配工具，能够：

1. **自动分析**输入音频的声音特征（F0、性别、音色、频谱等）
2. **智能匹配**预训练声模库（6个理论最优声模）
3. **精准对比**已训练的实际声模库（20个已训练声模）
4. **生成建议**提供详细的使用和训练建议

## 📊 功能特性

### 1. 声音特征分析

提取以下声音特征：

- ✅ **基频 (F0)**: 平均音高、标准差、范围
- ✅ **性别推测**: 男声/女声 + 置信度
- ✅ **音色类型**: 低音/中音/高音
- ✅ **频谱质心**: 音色亮度指标
- ✅ **共振峰**: 声道特征
- ✅ **音质评估**: 信噪比、剪辑率

### 2. 预训练声模匹配

与6个理论最优声模对比：

| 声模ID | 名称 | F0范围 | 适用场景 |
|--------|------|--------|----------|
| male_deep_001 | 男低音 - 深沉 | 80-200 Hz | 旁白、播音 |
| male_normal_001 | 男中音 - 标准 | 90-250 Hz | 对话、讲解 |
| male_high_001 | 男高音 - 明亮 | 110-320 Hz | 年轻角色、活力 |
| female_low_001 | 女低音 - 温柔 | 120-300 Hz | 抒情、安抚 |
| female_normal_001 | 女中音 - 标准 | 140-380 Hz | 对话、讲解 |
| female_high_001 | 女高音 - 清亮 | 170-450 Hz | 年轻角色、活泼 |

**匹配权重**:
- F0 均值: 40%
- F0 范围: 20%
- 性别: 15%
- 频谱质心: 15%
- 音色类型: 10%

### 3. 已训练声模匹配

与20个已训练的实际声模对比（speaker_0001 ~ speaker_0020）：

**匹配算法**:
- Embedding 余弦相似度: 50%
- F0 高斯相似度: 40%
- 基础分: 10%

### 4. 智能建议

根据匹配结果提供：

- ✅ **直接使用建议**: 如果已训练声模匹配度 >= 70%
- 📚 **训练建议**: 推荐预训练声模 + 训练参数
- 💡 **综合评估**: 对比理论和实际，给出最优方案

## 🎯 匹配度解读

### 预训练声模匹配度

| 匹配度 | 含义 | 建议 |
|--------|------|------|
| 80-100% | 极高匹配 | 使用该预训练模型训练效果最佳 |
| 60-80% | 高匹配 | 可以使用，训练时间可能稍长 |
| 40-60% | 中等匹配 | 可以使用，但需要更多训练 |
| <40% | 低匹配 | 建议考虑其他声模或跨性别训练 |

### 已训练声模匹配度

| 匹配度 | 含义 | 建议 |
|--------|------|------|
| 80-100% | 极高匹配 | 强烈推荐直接使用 |
| 70-80% | 高匹配 | 推荐使用，效果良好 |
| 60-70% | 中等匹配 | 可以使用，效果可接受 |
| <60% | 低匹配 | 建议训练新声模 |

## 🔧 高级选项

### 命令行参数

```
usage: voice_analysis_and_recommendation.py [-h] [-o OUTPUT] [--top-n TOP_N]
                                            [--rvc-logs RVC_LOGS]
                                            [--no-pretrained] [--no-trained]
                                            audio_path

positional arguments:
  audio_path            音频文件路径或目录

optional arguments:
  -h, --help            显示帮助
  -o, --output OUTPUT   输出报告文件路径（JSON）
  --top-n TOP_N         返回前 N 个匹配结果（默认: 3）
  --rvc-logs RVC_LOGS   RVC训练日志目录（默认: W:\rvc\logs）
  --no-pretrained       跳过预训练声模匹配
  --no-trained          跳过已训练声模匹配
```

### 自定义配置

#### 1. 修改预训练声模数据库

编辑 `voice_models_db.json`:

```json
{
  "voice_models": [
    {
      "id": "custom_voice_001",
      "name": "自定义声模",
      "voice_characteristics": {
        "f0_mean": 200.0,
        "f0_std": 30.0,
        ...
      }
    }
  ],
  "matching_weights": {
    "f0_mean": 0.40,
    "f0_range": 0.20,
    ...
  }
}
```

#### 2. 更新已训练声模数据库

删除 `trained_voices_db.json` 后重新运行，系统会自动扫描。

#### 3. 自定义 RVC 日志目录

```bash
python voice_analysis_and_recommendation.py audio.wav --rvc-logs "D:\MyRVC\logs"
```

## 📝 音频要求

### 推荐格式

- **格式**: WAV
- **采样率**: 40kHz（或其他高质量采样率）
- **位深度**: 16bit 或更高
- **时长**: 3-30秒（单个文件）

### 内容要求

- ✅ 清晰的说话声
- ✅ 无背景音乐
- ✅ 无明显噪音
- ✅ 自然的语调变化

### 批量分析建议

如果分析目录：
- 系统默认采样10个文件（可修改代码）
- 会计算平均特征
- 提高匹配准确性

## 🐛 故障排除

### 问题1: ModuleNotFoundError: No module named 'librosa'

**解决**:
```bash
pip install librosa
```

### 问题2: 找不到音频文件

**检查**:
- 路径是否正确（使用绝对路径）
- 文件是否存在
- 文件格式是否为 WAV

### 问题3: 匹配度都很低

**可能原因**:
- 音频质量差
- 特殊音色（儿童、老人）
- 性别跨越

**解决**:
- 提高音频质量
- 使用目录分析（多个样本）
- 考虑训练新声模

### 问题4: 已训练声模匹配度都是20%

**原因**: 数据库未更新

**解决**:
```bash
del trained_voices_db.json
python voice_analysis_and_recommendation.py audio.wav
```
