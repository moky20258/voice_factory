## 常用命令

### 生成音频
```bash
# 批量生成
python generate.py

# 质量筛选
python quality_filter.py

# DSP 增强
python dsp_enhancer.py
```

### 推荐系统
```bash
# 查询单个音频
python recommend_voice.py recording.wav

# 查询 Top 10
python recommend_voice.py recording.wav --top 10

# 批量查询
python recommend_voice.py --batch ./recordings

# 列出所有声模
python recommend_voice.py --list
```

### RVC 训练
```bash
# 训练所有 speaker
python rvc_batch_train.py --all

# 训练指定 speaker
python rvc_batch_train.py --speakers speaker_0001 speaker_0002
```

## 快速流程

### 从零开始
```
1. 准备文本 → voice_factory/texts/
2. 启动 API → python tools/api_server.py
3. 生成音频 → python generate.py
4. 质量筛选 → python quality_filter.py
5. DSP 增强 → python dsp_enhancer.py
6. 完整流水线 → python complete_pipeline.py
7. RVC 训练 → go-web.bat (在 RVC 目录)
8. 获得模型 → .pth + .index 文件
```

### 使用推荐系统
```
1. 录音 5-10 秒
2. python recommend_voice.py recording.wav
3. 获得 Top 5 推荐
4. 使用对应 speaker 的 RVC 模型
```
