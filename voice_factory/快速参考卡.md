# 🎤 声音分析与声模匹配 - 快速参考卡

## 🚀 快速启动

```bash
# 方式1: 双击批处理文件（推荐新手）
声音分析与匹配-交互版.bat

# 方式2: 命令行快速使用
python voice_analysis_and_recommendation.py audio.wav

# 方式3: 交互式版本
python interactive_voice_matcher.py

# 方式4: 观看演示
python demo_voice_matching.py
```

## 📋 常用命令

### 基础使用
```bash
# 分析单个音频
python voice_analysis_and_recommendation.py my_voice.wav

# 分析目录
python voice_analysis_and_recommendation.py ./my_audios

# 保存报告
python voice_analysis_and_recommendation.py my_voice.wav -o report.json
```

### 高级选项
```bash
# 返回5个匹配结果
python voice_analysis_and_recommendation.py my_voice.wav --top-n 5

# 只匹配预训练声模
python voice_analysis_and_recommendation.py my_voice.wav --no-trained

# 只匹配已训练声模
python voice_analysis_and_recommendation.py my_voice.wav --no-pretrained

# 自定义RVC日志目录
python voice_analysis_and_recommendation.py my_voice.wav --rvc-logs "D:\RVC\logs"
```

## 📊 输出解读

### 声音特征
```
F0 均值: 215.34 Hz          # 平均音高
推测性别: female            # 男/女声
音域范围: 180.45 Hz         # 音域跨度
频谱质心: 1920.45 Hz        # 音色亮度
```

### 匹配度
```
80-100%  极高匹配  ✅ 强烈推荐
60-80%   高匹配    ✅ 推荐使用
40-60%   中等匹配  ⚠️  可接受
<40%     低匹配    ❌ 建议训练新声模
```

## 🎯 使用场景

### 场景1: 快速变声
```bash
# 找到可直接使用的声模
python voice_analysis_and_recommendation.py my_voice.wav --no-pretrained
```

```bash

```

### 场景3: 完整分析
```bash
# 生成完整报告
python voice_analysis_and_recommendation.py my_voice.wav -o full_report.json
```

## 🔧 故障排除

### 问题: ModuleNotFoundError
```bash
pip install librosa
```

### 问题: 匹配度低
- 检查音频质量
- 使用目录分析（多个样本）
- 考虑训练新声模

### 问题: 已训练声模匹配度都是20%
```bash
del trained_voices_db.json
python voice_analysis_and_recommendation.py audio.wav
```

## 💡 提示

- ✅ 音频要求: WAV格式, 3-30秒, 清晰说话声
- ✅ 目录分析更准确（取平均特征）
- ✅ 匹配度>=70% 的已训练声模可直接使用
- ✅ JSON报告便于程序化处理
- ✅ 首次扫描声模库需要几秒，后续会缓存
