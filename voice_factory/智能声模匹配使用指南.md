# 🎯 智能声模匹配系统使用指南

## 📋 系统概述

声模匹配系统可以**自动分析目标说话人的声音特征**，从预训练声模库中**匹配最接近的声模**作为训练起点，从而：

- ✅ **减少训练时间** 60-80%
- ✅ **提升音质** 和相似度
- ✅ **降低失败率**
- ✅ **自动推荐** 最佳训练参数

2. 匹配最佳预训练声模
3. 根据匹配度推荐 epochs 数
4. 使用匹配的声模开始训练
5. 生成定制化模型文件

## 📊 声音特征说明

### **关键特征指标**

| 特征 | 说明 | 典型值 |
|------|------|--------|
| **F0 均值** | 基频平均值，决定音调高低 | 男: 100-180 Hz, 女: 180-300 Hz |
| **F0 范围** | 音域宽度，决定表现力 | 150-300 Hz |
| **频谱质心** | 音色明亮度 | 1000-2500 Hz |
| **共振峰** | 声道特征，决定音色 | F1: 300-1000 Hz, F2: 800-2500 Hz |

### **性别推断规则**

- F0 < 165 Hz → **男性**
- F0 ≥ 165 Hz → **女性**

### **音色分类**

| 类型 | F0 范围 | 说明 |
|------|---------|------|
| adult_deep | < 120 Hz | 男低音 |
| adult_normal | 120-165 Hz | 男中音 |
| adult_high | 165-200 Hz | 男高音/女低音 |
| female_normal | 200-250 Hz | 女中音 |
| female_high | > 250 Hz | 女高音 |

## ⚙️ 训练参数自动推荐

### **Epochs 推荐规则**

| 匹配度 | 音频时长 | 推荐 epochs | 说明 |
|--------|----------|-------------|------|
| ≥ 80% | ≥ 30 分钟 | 20-50 | 高匹配 + 充足数据 |
| ≥ 80% | < 10 分钟 | 30-60 | 高匹配 + 数据不足 |
| 60-79% | 任意 | 50-80 | 中等匹配 |
| < 60% | 任意 | 80-100 | 低匹配，需更多训练 |

### **Batch Size**

- 默认: **8**（适合大多数情况）
- GPU 显存充足: 可增加到 16
- GPU 显存不足: 降低到 4

## 💡 最佳实践

### **1. 准备训练数据**

- **最低要求**: 10 分钟高质量音频
- **推荐**: 30+ 分钟
- **音频质量**: SNR > 20 dB，无背景噪音
- **格式**: WAV, 40kHz, 16/24bit

### **2. 训练流程建议**

```
收集音频 (30+ 分钟)
    ↓
DSP 增强（complete_pipeline.py）
    ↓
智能匹配 + 训练（rvc_full_auto_train.py）
    ↓
测试音质
    ↓
如果不满意 → 调整参数重新训练
```

### **3. 常见问题**

**Q: 匹配度很低怎么办？**
A: 说明目标声音与现有声模差异较大，建议：
   - 增加训练 epochs（100-200）
   - 准备更多训练数据
   - 考虑从头训练（--no-match）

**Q: 训练后音质不好？**
A: 可能原因：
   - 训练数据不足 → 增加音频
   - epochs 不够 → 增加训练轮数
   - 音频质量差 → 重新筛选/增强

**Q: 如何添加自定义声模？**
A: 编辑 `voice_models_db.json`，添加新的声模条目。
